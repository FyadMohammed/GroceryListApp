## Features

### Core Features
- Add, edit, and delete grocery items  
- Mark items as purchased or unpurchased  
- Categorize items (fruits, vegetables, dairy, etc.)  
- Search and filter items  
- Persistent local storage  

### Optional (Future Enhancements)
- Multiple lists (e.g., Weekly, Party, Monthly)  
- Quantity and unit management  
- Notes or brand preferences  
- Collaborative lists (shared with family or friends)  
- Price tracking and total budget estimation  
- Voice input for adding items  
- Cloud sync with authentication
